# Dataset Manifest Examples

## CSV (`metadata.csv`)

```csv
audio_path,text
clips/001.wav,Hello world.
clips/002.wav,This is a voice dataset sample.
```

## JSONL (`metadata.jsonl`)

```jsonl
{"audio_path": "clips/001.wav", "text": "Hello world."}
{"audio_path": "clips/002.wav", "text": "This is a voice dataset sample."}
```
